# Boat and Yacht Charter Booking System for WordPress

[**Live Demo**](https://quanticalabs.com/wp_plugins/boat-charter-booking-system/) | [**Project page**](https://quanticalabs.com/wordpress-plugins/boat-charter-booking-system-for-wordpress/)

Boat & Yacht Charter Booking System is a powerful WordPress watercraft booking plugin for businesses of all sizes. It will work well in a small water equipment rental, e.g. renting rowing boats, kayaks or pedal boats, as well as the main booking system for marina service. The plugin will handle a large number of vessels and an unlimited number of locations.

It provides a simple, step-by-step booking process with online payments, e-mail and sms notifications, WooCommerce and Google Calendar integration and an intuitive backend administration.
